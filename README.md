# yandexlyceum-pyqt
Проект по технологии PyQT для Лицея Академии Яндекс

Проект представляет из себя приложения для хранения домашней электронной библиотеки.
# Используемые библиотеки:
- PyQT5
- sqlite3
- os
- hashlib
- requests
- ...

# Функции приложения:
- Хранение файлов книг(fb2, pdf, epub)
- ...
